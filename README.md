# s3rv3rl3ss

Website as an information center for serverless services.

## Tech Stack

- Vue 3 + Vite
- Tailwind CSS
- Vue Router
- Fuse.js (fuzzy search)
- AWS Amplify (hosting)

## Development

```bash
nvm use 20
npm install
npm run dev
```

## Adding Services

Edit `src/data/services.json` and commit.

## Deploy

Connected to AWS Amplify — auto-deploys on push.
